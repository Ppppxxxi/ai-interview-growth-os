# v0.2 Release Notes

## 版本定位

v0.2 是 AI 面试成长 OS 的 **Personal MVP**：面向单个求职用户的本地可用版本。

它不是普通模拟面试工具，也不是完整商业化 SaaS。当前版本聚焦一个高频真实场景：用户已经在 ChatGPT、Claude、Codex 或其他 AI 工具里做过模拟面试，需要把分散的问答、点评和复盘沉淀成下次面试可复用的回答资产。

## 本版本完成了什么

### 1. 岗位工作台

- 新建、编辑、删除岗位档案。
- 保存岗位 JD、公司、岗位名、方向和阶段。
- 删除岗位时同步清理关联面试记录、复盘和回答资产。
- 粘贴外部 AI 模拟面试对话。
- 预览并手动编辑解析出的原问题、原回答、追问和 AI 点评。
- 生成结构化复盘和优化回答。
- 保存多场面试记录，并支持记录切换、详情查看和删除。
- 提供示例岗位和示例面试对话，首次试用可快速跑通完整流程。

### 2. 回答资产库

- 将优化回答保存为回答资产。
- 支持关键词搜索。
- 支持按岗位方向、问题类型、能力短板、来源岗位、置信度和使用状态筛选。
- 每条资产保留来源岗位、来源面试、原问题、原回答、问题点、优化回答、适用问题和使用建议。
- 支持记录使用反馈：使用结果、使用岗位、关联面试、面试官追问和使用后复盘。
- 支持回答版本管理：保存新版本、查看历史版本、恢复历史版本为当前推荐回答。

### 3. Markdown 导出

- 导出单场面试复盘。
- 导出岗位考前准备包。
- 导出当前筛选下的回答资产合集。
- 导出单条回答资产。
- 导出内容包含来源信息、使用反馈和历史回答版本。

### 4. 本地持久化

- 使用浏览器 `localStorage` 保存岗位档案、面试记录、复盘、回答资产、使用反馈和回答版本。
- 不需要登录、后端服务或 API Key。
- 真实 JD、真实面试对话和个人复盘只保存在用户本地浏览器，不进入 Git 仓库。

## 当前仍不做什么

- 不接真实 LLM。
- 不做登录和云同步。
- 不做多人协作。
- 不做求职 CRM、自动投递、岗位搜索或薪资谈判。
- 不做完整能力图谱或复杂统计分析。
- 不承诺跨行业、跨岗位的泛化复用，优先证明同岗位多轮和同方向类似岗位复用。

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev
```

验证：

```powershell
npm.cmd test
npm.cmd run build
```

## 发布前检查摘要

- 仓库不需要 `.env` 或任何 API Key。
- `.gitignore` 已忽略 `node_modules/`、`dist/`、`.env*` 和 TypeScript 构建缓存。
- 当前样例数据是 mock 数据，不包含真实求职隐私。
- 如果后续用真实 JD 或真实面试对话截图，请在公开前脱敏。
