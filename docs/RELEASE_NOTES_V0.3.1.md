# v0.3.1 Release Notes

## 版本定位

v0.3.1 是真实使用校准版。它不继续增加新功能，重点解决 v0.3 发布后暴露出的体验问题：空首页很清楚，但进入工作台后信息重新变得复杂，用户会从线性任务进入三栏后台。

## 本版本改了什么

- 首页文案与主流程统一为“粘贴材料 -> 检查草稿 -> 保存回答”。
- 顶部导航从功能术语改为任务语言：
  - 生成本场回答
  - 我的回答库
  - 面试复盘
- 工作台首屏聚焦“导入一整份面试材料”。
- 增加三步流程提示：粘贴材料、检查草稿、保存回答。
- 标准 JD + 模拟面试对话入口降级为“高级导入”折叠区。
- 新建岗位、岗位编辑默认折叠，减少表单压力。
- 未生成前右侧只展示下一步提示，不再展示 0 场记录、待确认或待生成类状态。
- 生成后再展示面试记录、回答、岗位准备、模拟练习和下次可能被问。

## 当前仍不做什么

- 不接真实 LLM。
- 不重写三 Tab 信息架构。
- 不做完整 wizard。
- 不做登录和云同步。
- 不把回答资产概念完全从产品中移除，只在主流程文案里优先使用“下次可用回答”。
