# v0.2.1 Release Notes

## 版本定位

v0.2.1 是 AI 面试成长 OS 的真实新用户体验修正版。它不继续扩展功能，重点解决首次打开页面时的信息过载、示例数据困惑和核心流程不清晰。

## 本版本改了什么

- 新增首次进入页，提供“开始准备我的面试”和“使用示例体验”两个入口。
- 默认不再把示例数据直接写入浏览器本地数据。
- 示例岗位在工作台中显示“示例数据”标识。
- 示例回答资产默认状态改为“待实战验证”。
- 增加“恢复示例”和“清空本地数据”入口。
- 空白岗位下隐藏草稿复盘和草稿回答，改为清晰的空状态提示。
- 输入框补充 JD 和外部面试对话的格式提示。
- “解析结果确认”改为“检查问题和回答是否识别正确”，降低用户理解成本。
- 导航文案调整为“准备本场面试 / 可复用回答 / 复盘总结”。
- 移动端把主流程放在岗位列表和新建岗位表单之前。

## 当前仍不做什么

- 不接真实 LLM。
- 不做登录和云同步。
- 不做完整引导 wizard。
- 不做复杂任务管理。
- 不做求职 CRM 或自动投递。
