# v0.3.2 Release Notes

## 版本定位

v0.3.2 是入口分流版。它修正 v0.3.1 仍然偏“折叠工作台”的问题：首次用户不应该先进入工作台再寻找入口，老用户也不应该每次都被打断到开始页。

## 本版本改了什么

- 新增独立 `FocusFlowPage`，专门承载一次面试材料导入任务。
- 无本地真实数据时，默认进入聚焦流程页。
- 有历史岗位、面试记录或回答时，默认进入工作台。
- 顶部导航增加“开始新面试”，老用户可以随时从工作台进入聚焦流程。
- 工作台导航改为“工作台 / 我的回答库 / 面试复盘”，不再把工作台命名为生成流程。
- 工作台移除材料导入表单，只保留历史记录、回答、复盘和训练建议等管理内容。
- 聚焦流程固定为：
  - 粘贴 JD、面试记录或模拟面试对话
  - 解析材料
  - 确认识别结果
  - 生成下次可用回答
  - 保存到我的回答库
- 新增 `hasRealWorkspaceData` 判断，避免空白占位岗位被当作真实历史数据。

## 当前仍不做什么

- 不接真实 LLM。
- 不做登录和云同步。
- 不做完整多页 wizard。
- 不做文件上传。
- 不把工作台删除；工作台仍是长期沉淀管理区。
